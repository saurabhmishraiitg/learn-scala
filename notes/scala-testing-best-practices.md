# Scala Testing Best Practices

- Inherit a common Spec class which has relevant imports. You can do all the relevant initializations in the parent class, avoiding duplication of code
  - e.g. **parent spec class**

  ```scala
  import com.holdenkarau.spark.testing.DataFrameSuiteBase
  import com.typesafe.scalalogging.StrictLogging
  import com.walmart.archetype.core.utils.ConfigUtil
  import com.walmart.archetype.core.utils.LibraryEnum.LibraryEnum
  import com.walmart.dsi.gg.util.WorkflowRunnerConfigureUtil
  import com.walmart.dsi.gg.utils.SparkIOUtil
  import org.apache.spark.sql.DataFrame
  import org.scalatest.{BeforeAndAfterAll, FlatSpec, Matchers}

  class BaseSpec extends FlatSpec with DataFrameSuiteBase with BeforeAndAfterAll with StrictLogging with Matchers {

    var libraryObjects: Option[Map[LibraryEnum, Any]] = None
    var paramsMap: Map[String, Any] = Map.empty
    var dataFrameMap: Map[String, DataFrame] = Map.empty

    override def beforeAll {
      // Need to invoke the beforeAll implementation from "DataFrameSuiteBase" class to ensure sparkSession is loaded
      BaseSpec.super.beforeAll

      // Local run specific params
      paramsMap = Map("runtype" -> "standalone", "runmode" -> "local", "env" -> "test", "workflow" -> "TemplateWorkflow")
      paramsMap = WorkflowRunnerConfigureUtil.updateParamMapGCSAuto(paramsMap, ConfigUtil.appConfig)

      logger.debug(s"update paramsMap [$paramsMap")

      // Setting the sparkSession generated by DataFrameSuiteBase in the SparkArchetype context
      SparkIOUtil.spark = spark
    }
  }
  ```

  - e.g. **child spec class**

  ```scala
  class TemplateExtractorSpec extends BaseSpec {

    "TemplateExtractor" should "extract relevant dataframes" in {
    }
  }
  ```

- If you want to ignore some test case run during the automated build of your project, but available to run manually local, leverage `tags`

  - e.g. **tag class**

      ```scala
      import org.scalatest.Tag

      object LocalOnlyTests extends Tag("com.walmart.dsi.gg.LocalOnlyTests")
      ```

  - e.g. usage of **tag class**

  ```scala
  import com.walmart.dsi.gg.LocalOnlyTests

  "local run of OnPremToGCS with inputPath CSV data" should "overwrite data as ORC in GCS bucket" taggedAs LocalOnlyTests in {
  }
  ```

  - e.g. relevant exclusion in **pom.xml**

  ```xml
      <build>
          <plugins>
              <!-- enable scalatest -->
              <plugin>
                  <groupId>org.scalatest</groupId>
                  <artifactId>scalatest-maven-plugin</artifactId>
                  <version>${scalatest.maven.plugin.version}</version>
                  <configuration>
                      <tagsToExclude>com.walmart.dsi.gg.LocalOnlyTests</tagsToExclude>
                  </configuration>

                  <executions>
                      <execution>
                          <id>test</id>
                          <goals>
                              <goal>test</goal>
                          </goals>
                      </execution>
                  </executions>
              </plugin>
          </plugins>
      </build>
  ```
